# CyberBulling_DNN
A Image Classification for Bulling, No Bulling, built on Top of MobleiNet Transfer Learning

### To run the project 

First Install all the packages by running the command
```
pip install -r requirements.txt
```

then run the app.py file

```
python app.py
```
